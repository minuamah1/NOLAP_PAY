# NOLAP_PAY

##Author
Nuamah Maymuna Ishaak

##Description
A simple Python project that simulates NOLAP Pay system.

##How to run
1. Download this repository to your computer.
2. Make sure Python 3 is installed on your computer.
3. Open the project folder in VS Code or any terminal.
4. Run the following command:
   python nolap_pay_project.py
5. Follow the on-screen instructions to use NOLAP pay
